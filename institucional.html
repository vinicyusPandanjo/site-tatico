Segurança e Privacidade

POLÍTICA DE PRIVACIDADE DO USUARIOS E CLIENTES DO NAUTIKA COMERCIAL DE ARTIGOS PARA LAZER LTDA E NAUTIKA COMERCIO ELECTRONICO LTDA (“NAUTIKA”)

1. BASE LEGAL PARA O TRATAMENTO:

Esta política de privacidade foi elaborada pela NAUTIKA COMERCIAL DE ARTIGOS PARA LAZER LTDA. e NAUTIKA COMÉRCIO ELETRÔNICO LTDA. (doravante "NAUTIKA"), de acordo com a Lei Federal nº 12.965, de 23 de abril de 2014 (doravante "Marco Civil de Internet") e Lei Federal nº 13.709, de 14 de agosto de 2018 (Lei Geral de Proteção de Dados, a partir de agora "LGPD"). O site NAUTIKA (www.nautikalazer.com.br) busca cumprir as regras estabelecidas na LGPD, bem como respeitar os princípios previstos no artigo seis da referida Lei: finalidade, adequação, necessidade, livre acesso, qualidade dos dados, transparência, segurança, prevenção, não discriminação e responsabilização.

2. CONTROLADOR.

A NAUTIKA é responsável pelo tratamento dos dados pessoais dos titulares. Assim, as informações de contato dessa entidade:
  • Nome do Controlador: NAUTIKA COM. DE ARTIGOS PARA LAZER LTDA E NAUTIKA COMERCIO ELETRÔNICO LTDA.
  • Endereço: Rua Santana de Ipanema, 597 - Cumbica - Guarulhos/SP.
  • E-mail: data-privacy@merama.io
  • Telefone: (11) 2085-4622.

3. RESPONSÁVEL.

Para tudo relacionado ao site NAUTIKA, o responsável por atuar como um canal de comunicação entre o Controlador de Dados, os Titulares e a Autoridade Nacional de Proteção de Dados, é o Departamento de Proteção de Dados Pessoais do NAUTIKA.O Titular pode entrar em contato com o Controlador de Dados através do endereço de e-mail data-privacy@merama.io, a fim de resolver quaisquer dúvidas sobre esta política de privacidade, ou para obter mais informações sobre o tratamento de dados realizado pelo NAUTIKA com base na LGPD.

4. DIREITOS DOS TITULARES DE DADOS PESSOAIS.

O Titular dos dados pessoais tem os seguintes direitos, que são conferidos pela LGPD:
  • Direito de confirmação e acesso (Art. 18, pontos I e II): obter confirmação quanto ao tratamento ou não de seus dados pessoais e, se for o caso, o direito de acessá-los;
  • Direito à retificação (Art. 18, inciso III): solicitar a correção de dados pessoais incompletos, imprecisos ou obsoletos;
  • Direito de limitar o tratamento de dados (Art. 18, ponto IV): limitar o tratamento de seus dados pessoais, podendo exigir a exclusão de dados desnecessários, excessivos ou tratados em violação às disposições da LGPD;
  • Direito de objeto (Art. 18, § 2º): opor-se a qualquer momento ao tratamento de dados pessoais por razões relacionadas à sua situação particular, com base em uma das hipóteses de retirada de consentimento ou em caso de descumprimento das disposições da LGPD;
  • Direito à portabilidade de dados (Art. 18, ponto V): realizar a portabilidade de seus dados pessoais a outro prestador de serviços ou produtos, mediante solicitação expressa, de acordo com as normas da autoridade nacional, de acordo com segredos comerciais e industriais;
  • Direito de não estar sujeito a decisões automatizadas (Art. 20): solicitar uma revisão das decisões tomadas exclusivamente com base no tratamento automatizado de dados pessoais relacionados aos seus interesses, incluindo decisões destinadas a definir seu perfil pessoal, profissional, de consumo e de crédito ou aspectos de sua personalidade.

5. QUAIS DADOS PESSOAIS SÃO TRATADOS?

O uso de certos recursos do site NAUTIKA pelo Titular dependerá do tratamento dos seguintes dados pessoais: Dados pessoais dos Usuários:
  • Dados de identificação.
  • Dados de autenticação.
  • Dados de Contato.
  • Dados de navegação (cookies).
Dados pessoais dos Clientes:
  • Dados de identificação.
  • Dados de autenticação.
  • Dados de Contato.
  • Dados de navegação (cookies).
  • Dados patrimoniais ou financeiros.
